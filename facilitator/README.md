# Facilitator pack — Research Supercharged

Everything you need to **run** the session (the *how*, *when*, and *why*). Provisioning the
Foundry project and granting participant access is a separate, one-time admin job — see
**[../admin/](../admin/)**.

## What's here

| Doc | Use it for |
|-----|------------|
| **[facilitator-guide.md](./facilitator-guide.md)** | The **run-of-show**: pre-flight checklist, minute-by-minute script, demo cues, contingencies, and common pitfalls. Your primary doc on the day. |
| **[workshop-plan.md](./workshop-plan.md)** | The **design rationale**: learning goals, why one shared project, the two-rail model, Foundry feature mapping, risks & fallbacks. Read once before you run it. |
| **[slides-outline.md](./slides-outline.md)** | A **deck skeleton** (~40 slides) — section dividers, teach slides, and "hands-on hold" slides. Build the real deck from this. |

## Run order

1. **Before the day** — an admin provisions the shared project and assigns access:
   **[../admin/](../admin/)** (Basic Foundry account, model deployments, **Foundry User** RBAC).
2. **Prep** — read [workshop-plan.md](./workshop-plan.md), then work the **pre-flight checklist** at
   the top of [facilitator-guide.md](./facilitator-guide.md) (dry-run all four lab scripts).
3. **On the day** — drive from [facilitator-guide.md](./facilitator-guide.md); project the deck you
   built from [slides-outline.md](./slides-outline.md).

## Schedule at a glance (≈ 3 hours)

| Time | Lab | Feature |
|------|-----|---------|
| 0:00 – 0:20 | 0 · Hello, Research Copilot | Agents |
| 0:20 – 0:55 | 1 · Search the literature | Web Search (`model-router`) |
| 0:55 – 1:35 | 2 · Ground on your papers | File Search / RAG |
| 1:35 – 1:45 | ☕ Break | — |
| 1:45 – 2:25 | 3 · Analyse the data | Code Interpreter |
| 2:25 – 2:50 | 4 · Add a tool | Function calling / MCP |
| 2:50 – 3:00 | 5 · Take it home | Responsible AI + next steps |
| _optional_ | 🌟 Bonus · Evaluate & observe | Evaluations (portal + SDK) — [labs/bonus/](../labs/bonus/) |

> Trim to **2.5 hours** by tightening Lab 2's work time and running **Lab 4 as a demo** — see the
> contingencies in [facilitator-guide.md](./facilitator-guide.md). The **bonus evaluation labs**
> are self-paced for fast finishers or post-workshop — not part of the timed run.
