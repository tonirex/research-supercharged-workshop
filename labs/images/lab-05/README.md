# Screenshots for Lab 5 (Portal Walkthrough)

These images are referenced by [`../../lab-05-portal.md`](../../lab-05-portal.md). Each was captured
from the **main content pane** of the Foundry portal ([ai.azure.com](https://ai.azure.com)) with the
browser chrome cropped out.

| File | Step | What it shows |
|------|------|---------------|
| `01-delete-agent.png` | 1 | The **Agents** list with the **`rc-ac`** row's **Actions (⋮)** menu open, showing the single **Delete** item. Only your own `rc-<initials>` agents should be removed — this is a shared project. |
| `02-delete-confirm.png` | 2 | The **Permanently delete "rc-ac"?** dialog. You must **type the agent's name** to confirm; the **Delete** button stays disabled until it matches, and **Cancel** backs out safely. |

> Capture tip: for step 1, screenshot the portal's main content region so the left nav and the open
> **Actions** menu are both visible. For step 2, screenshot **just the confirmation dialog** so the
> type-to-confirm guardrail reads as a tight crop.
